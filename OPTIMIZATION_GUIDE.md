## 🚀 使用場景範例

### 場景1：「台積電現在值不值得買？」
**原有版本回答：**
- 短線技術面（支撐、壓力、KD指標）
- 籌碼面（三大法人）
- 短線目標價 & 停損位

**優化版本回答：**
+ ✓ 同上（短線分析保留）
+ 💼 **基本面補充** — 財報、毛利率、客戶集中度、地緣政治風險
+ 📊 **市場環境** — 當前是科技多頭/空頭，適合買入嗎？
+ 🎯 **分級建議**：
  - 激進者：短線突破買進
  - 平衡者：中線突破 + 分批買進
  - 保守者：觀望或長期分批布局
+ ⏰ **時間框架明確** — 短線/中線/長期預期

### 場景2：「最近有什麼值得關注的新興市場機會？」
**新增能力完全展開：**
- 🌍 掃描全球新興市場環境（印度、巴西、東盟）
- 📈 各市場當前景氣判斷（多頭/整理/空頭）
- 💡 識別主題機會（消費升級、製造業北遷、大宗商品）
- 💰 推薦個股 + 催化劑 + 風險評估
- 💱 匯率風險提示

### 場景3：「俄烏戰爭對歐洲股市的長期影響？」
**新增能力：**
- 🔗 **事件影響鏈條** — 能源→通脹→央行升息→股市下跌
- ⏰ **時間維度** — 瞬間恐慌 vs 中期重估 vs 長期結構變化
- 🏢 **受益/受害行業清單** — 能源股上升，製造業受損
- 📊 **宏觀背景** — ECB政策、德國經濟、歐盟政策
- 🎯 **操作建議** — 短線應做什麼、中線布局什麼

### 場景4：「我想要高股息投資組合，全球有什麼推薦？」
**組合分析能力（新增）：**
1. 掃描全球高股息資產（巴西5%+、印度銀行、英美金融）
2. 評估匯率風險（新興市場貨幣貶值風險）
3. 評估政治與經濟風險（集中度、分散度）
4. 提供配置建議（地區配置、行業配置）
5. 定期監控機制

## 💡 核心改進點

### 1. **從「短線」到「全方位」**
原有短線交易導向 → 現在支援全類型投資者的分析

```
短線交易者   ━━━ 保留原有的快速進出、技術面、籌碼
           ┃
一般投資者   ━━━ 新增基本面、估值、催化劑分析
           ┃
長期投資者   ━━━ 新增產業趨勢、競爭力、增長空間
           ┃
全球投資者   ━━━ 新增匯率、政治、宏觀風險評估
```

### 2. **從「台美」到「全球」**
台灣 + 美國（2個市場）→ 全球12個主要市場 + 加密資產

- 原有市場：**深化分析**（市場文檔保留最佳實踐）
- 新增市場：**快速上手**（新建文檔提供完整框架）

### 3. **從「有據可依」到「來源透明」**
保持原有的「必須檢附資料來源」原則，並擴展到全球市場

- 每個分析都有web_search支撐
- 數據來源明確標注
- 資料時效性（最新新聞、實時價格）

### 4. **從「風險提示」到「風險量化」**
不只說「風險高」，而是：

| 原有 | 優化後 |
|------|--------|
| 「風險高」 | 「風險評級：中高，主要源自政治不確定性(±30%)與匯率波動(盧比年貶值5-10%)」 |
| 「監管風險」 | 「政策風險評級：高，2024年選舉變數、上屆政府科技監管政策可能反轉」 |
| 「有下跌風險」 | 「下跌風險30%至$280，主要驅動因素：Fed升息超預期、中國景氣再次放緩」 |

## 📋 使用檢查清單

### 提供分析前的必做項目
- [ ] **搜尋最新數據**（價格、新聞、大盤）
- [ ] **評估宏觀環境**（Fed、通脹、大盤景氣）
- [ ] **判斷市場環境**（多頭/整理/空頭）
- [ ] **確認資料來源**（至少主要數據點註記來源）
- [ ] **標注投資風格**（適合誰：激進/平衡/保守）
- [ ] **清晰標注時間框架**（短線/中線/長期）
- [ ] **明確列出風險與停損**（數字化風險評估）
- [ ] **包含免責聲明**（「僅供參考，非投資建議」）

### 特殊情況的警示
| 情況 | 警示等級 | 建議行動 |
|------|--------|---------|
| VIX > 30 (極端恐慌) | 🔴 極高 | 建議暫停操作、增加現金 |
| 政治風險突發 | 🔴 高 | 標注「政治風險倍增」 |
| 估值泡沫明顯 | 🟠 中高 | 「估值風險，不適合新手」 |
| 新興市場匯率崩潰 | 🔴 高 | 「匯率風險超過基本面收益」 |
| 加密資產波動 | 🔴 極高 | 「極高風險，嚴禁杆桿」 |

**Q: 優化版本會影響短線交易的精準度嗎？**

**Q: 加密資產分析靠譜嗎？**  
A: 和股票分析一樣靠譜，但會明確標注「極高風險」。建議普通投資者只配置資產的1-3%。

**Q: 為什麼要檢附資料來源？**  
A: 防止提供過時或錯誤資訊，確保投資決策基於最新、可驗證的數據。
