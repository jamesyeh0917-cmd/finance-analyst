## 一、三大法人籌碼解讀

### 概念說明
台股三大法人指：**外資（QFII）、投信（本土基金）、自營商**，其買賣超資料每日收盤後公告，是判斷主力動向的核心依據。

### 外資（外國機構投資者）
- **影響力最大**，持有台股市值約 40%，對大型權值股（台積電、聯發科等）有決定性影響
- 外資連買 → 通常伴隨指數上漲，尤其大盤權值股
- 外資連賣 → 需警惕，可能是趨勢反轉訊號
- 查詢來源：https://www.twse.com.tw/zh/trading/fund/T86.html

| 訊號 | 意涵 | 操作建議 |
|------|------|---------|
| 外資連買 5 日以上 | 多頭趨勢強化 | 積極佈局 |
| 外資單日大賣 >100 億 | 短線警示 | 降低倉位，觀察次日 |
| 外資賣超但股價不跌 | 內資接手，籌碼穩定 | 持股觀察 |
| 外資買超但股價不漲 | 可能有反向賣壓 | 謹慎，不追高 |

### 投信（本土基金公司）
- 主導**中小型成長股**，護盤力強
- 投信連買是中線偏多訊號（有績效壓力，不輕易賣出）
- 投信月底前容易護盤（淨值結算壓力）
- **重點觀察**：近 10 日累積買超超過股本 1% 的個股，往往是強勢標的

### 自營商（券商自有資金）
- 資金規模較小，但反應快速，常是短線風向球
- 主要看**自行買賣**部位，「避險」部位意義不大
- 自營買超 + 外資買超 = 籌碼同向，訊號更可信

### 三大法人合計解讀矩陣

| 外資 | 投信 | 自營 | 解讀 |
|------|------|------|------|
| 買 | 買 | 買 | 🟢 全面多頭，強力做多訊號 |
| 買 | 買 | 賣 | 🟢 多頭為主，自營避險屬正常 |
| 買 | 賣 | 買 | 🟡 留意投信出脫壓力 |
| 賣 | 買 | 買 | 🟡 內資接手，短線觀望 |
| 賣 | 賣 | 賣 | 🔴 三方賣超，迴避做多 |
| 賣 | 買 | 賣 | 🟡 投信護盤，多空分歧，縮手 |

## 二、台指期貨分析

### 未平倉口數（OI）解讀

| 狀況 | 意涵 |
|------|------|
| 未平倉增加 + 期貨上漲 | 多方積極建倉，多頭趨勢 |
| 未平倉增加 + 期貨下跌 | 空方積極建倉，空頭趨勢 |
| 未平倉減少 + 期貨上漲 | 空方回補，反彈力道有限 |
| 未平倉減少 + 期貨下跌 | 多方停損，跌勢可能持續 |

### 三大法人期貨部位
- **外資期貨淨多單**：最重要的領先指標
  - 淨多單增加 → 做多訊號
  - 淨空單擴大 → 做空訊號，需特別警惕
- **自營商期貨**：通常為避險，反向操作，參考性較低

### 期現貨價差（基差）
- 正價差（期貨 > 現貨）→ 市場偏多預期
- 負價差（期貨 < 現貨）→ 市場偏空或法人避險
- 結算前正價差收斂 → 多方獲利了結壓力

查詢來源：https://www.taifex.com.tw/cht/3/futContractsDate

## 三、台股選擇權分析

### Put/Call Ratio（P/C 比）
- P/C > 1.5：市場極度悲觀 → **逆向做多訊號**（恐慌過度）
- P/C < 0.8：市場極度樂觀 → **逆向做空訊號**（過熱）
- 0.8 ~ 1.2：正常範圍，以其他指標為主

### 最大痛點（Max Pain）
- 讓最多選擇權買方虧損的結算價位
- 結算前 1-2 週，指數有向最大痛點靠攏的傾向
- 可作為結算週的區間操作參考

### 隱含波動率（IV）
- IV 急升 → 市場恐慌，買 Put 成本高，此時不宜加碼買保護
- IV 低位 → 市場平靜，是**買入選擇權的好時機**
- 台股 TVIX 高於 25 → 通常接近短線底部，可考慮逆向佈局

查詢來源：https://www.taifex.com.tw/cht/3/optContractsDate

## 四、台股技術面特有觀察

### 台積電權重效應
- 台積電（2330）佔加權指數權重約 30%
- **台積電走勢幾乎決定大盤方向**，分析大盤必須確認台積電同向

### 融資融券解讀

| 狀況 | 意涵 |
|------|------|
| 融資增加 + 股價上漲 | 散戶追漲，後續空間有限，注意反轉 |
| 融資減少 + 股價上漲 | 法人主導，健康多頭 |
| 融券增加 | 市場看空，也可能是軋空燃料 |
| 融資大量強制斷頭 | 恐慌賣壓結束，往往是反彈起點 ✅ |

- 融資使用率 > 50% 的個股，做多需謹慎

### 台股盤後資訊解讀順序（每日必看）
1. 大盤收盤漲跌幅 + 成交量
2. 台積電當日走勢
3. 三大法人買賣超（總計 + 個股明細）
4. 台指期夜盤走勢（反映隔日開盤預期）
5. 美股期貨走勢（ES、NQ）

## 七、台股操作心法

1. **跟著外資走**：外資連續買超的個股是最安全的多方標的
2. **投信抱股不輕易賣**：投信買入後中線持有穩定，適合波段跟進
3. **量先價行**：放量通常領先股價突破 1-2 天，留意縮量後的爆量
4. **台積電定江山**：台積電多空決定約 30% 大盤方向，必看
5. **結算前回歸最大痛點**：期指結算前，指數有向最大痛點靠攏的傾向
6. **融資斷頭是底部訊號**：大量強制斷頭後，恐慌賣壓結束，往往是反彈起點
7. **外資期貨淨多單是領先指標**：比現貨籌碼更早反映外資真實方向
8. **籌碼集中才能漲**：散戶主導的股票漲幅有限；法人主導的股才有大行情
