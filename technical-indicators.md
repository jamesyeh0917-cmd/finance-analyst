# technical-indicators.md — 技術指標詳細說明

> 本文件為 `SKILL.md` 的技術分析延伸，提供各主要技術指標的完整說明與短線操作應用。
> 技術指標是輔助工具，應結合籌碼面與基本面使用，單獨依賴技術指標容易有盲點。

---

## 一、均線系統（Moving Averages）

### 常用均線週期

| 均線 | 週期 | 用途 |
|------|------|------|
| MA5 | 5 日 | 超短線方向，日內參考 |
| MA10 | 10 日 | 短線趨勢，2 週操作週期 |
| MA20 | 20 日 | 短中線支撐/壓力，月線 |
| MA60 | 60 日 | 中線趨勢，季線 |
| MA120 | 120 日 | 半年線，中長期方向 |
| MA240 | 240 日 | 年線，長期趨勢 |

### 均線排列解讀

**多頭排列**：MA5 > MA10 > MA20 > MA60（短均線在長均線之上）
- 趨勢向上，回測短均線是好的買點
- 均線間距越大，多頭動能越強

**空頭排列**：MA5 < MA10 < MA20 < MA60（短均線在長均線之下）
- 趨勢向下，反彈到短均線是好的賣點
- 均線間距越大，空頭動能越強

**均線糾結**：各均線密集交叉
- 趨勢不明確，等方向確認後再進場
- 通常是盤整後的方向選擇前兆

### 均線操作要點
- **黃金交叉**：短均線（MA5/MA10）向上穿越長均線（MA20/MA60）→ 做多訊號
- **死亡交叉**：短均線向下穿越長均線 → 做空訊號
- **均線扣抵**：K 線站上所有均線後，均線「扣抵」低點，均線自然上彎 → 多頭加速
- **均線乖離率**：股價遠高於均線（乖離過大）→ 短線回調風險；遠低於均線 → 超跌反彈機會

## 二、KD 指標（隨機指標 / Stochastic Oscillator）

### 計算基礎
KD 指標基於最近 N 天（通常 9 天）的高低點範圍，反映當前收盤價在此區間的相對位置。
- **K 值**：快線（對 D 值更敏感）
- **D 值**：K 值的移動平均（慢線）

### 數值意涵

| 區間 | 意涵 |
|------|------|
| K/D > 80 | 超買區，短線可能回調 |
| K/D < 20 | 超賣區，短線可能反彈 |
| K/D 在 50 附近 | 中性，趨勢不明 |

### 訊號解讀

**黃金交叉（做多訊號）**：K 值從下方向上穿越 D 值
- 發生在超賣區（<20）→ 最強的做多訊號
- 發生在中性區（50 附近）→ 中等可信度

**死亡交叉（做空訊號）**：K 值從上方向下穿越 D 值
- 發生在超買區（>80）→ 最強的做空訊號
- 發生在中性區 → 中等可信度

**背離**（Divergence）：
- **多頭背離**：股價創新低，但 KD 未創新低 → 下跌動能衰竭，反彈訊號
- **空頭背離**：股價創新高，但 KD 未創新高 → 上漲動能衰竭，拉回訊號

### 短線應用注意
- **強勢股的鈍化**：趨勢強勁時，KD 可在超買區長期鈍化（>80 持續多日）→ 不能盲目做空
- **弱勢股的鈍化**：KD 在超賣區鈍化（<20 持續多日）→ 不能盲目做多
- 強趨勢中以「8/20 黃金交叉」（日 KD 黃金交叉後）進場，停損設前低

## 三、RSI 指標（相對強弱指數）

### 計算基礎
RSI 衡量一段時間內漲幅與跌幅的比值，反映多空力量對比。
- 通常使用 14 日 RSI

### 數值意涵

| 區間 | 意涵 |
|------|------|
| RSI > 70 | 超買，短線過熱 |
| RSI < 30 | 超賣，短線恐慌 |
| RSI 50 | 多空分界線，站上偏多，跌破偏空 |
| RSI > 80 | 強勢股特有，趨勢延續中 |
| RSI < 20 | 極度超賣，恐慌底部 |

### 訊號解讀

**RSI 50 分界線**：
- RSI 突破 50 向上 → 趨勢由空轉多，做多訊號
- RSI 跌破 50 向下 → 趨勢由多轉空，做空訊號

**背離應用**（RSI 最重要的應用）：
- **多頭背離**：股價創新低 + RSI 未創新低 → 底部反轉訊號（最強訊號之一）
- **空頭背離**：股價創新高 + RSI 未創新高 → 頂部反轉訊號

**失敗擺動（Failure Swing）**：
- RSI 從超賣反彈突破前高 → 強力做多訊號
- RSI 從超買下跌跌破前低 → 強力做空訊號

### 多週期 RSI 應用
- **週 RSI + 日 RSI 共振**：週 RSI 超賣 + 日 RSI 出現黃金交叉 → 極強反彈訊號
- 短線操作主要看日 RSI，中線操作看週 RSI

## 四、MACD 指標

### 組成說明

| 元素 | 說明 |
|------|------|
| MACD 線（快線）| EMA12 - EMA26 |
| 訊號線（慢線）| MACD 線的 EMA9 |
| 柱狀圖（Histogram）| MACD 線 - 訊號線，反映動能強弱 |
| 零軸 | MACD 線在零軸上方偏多，下方偏空 |

### 核心訊號

**黃金交叉**：MACD 線向上穿越訊號線
- 發生在零軸以下 → 中線反轉訊號（最強）
- 發生在零軸附近 → 中等訊號
- 發生在零軸以上 → 中線延續做多訊號

**死亡交叉**：MACD 線向下穿越訊號線
- 發生在零軸以上 → 中線反轉訊號（最強）
- 發生在零軸以下 → 延續做空訊號

**柱狀圖解讀**：
- 柱狀圖由負轉正 → 多方動能出現
- 柱狀圖持續縮短（趨近零）→ 動能衰竭，注意趨勢反轉
- 柱狀圖放大 → 趨勢加速

### MACD 背離（最重要應用）

**多頭背離**：股價創新低，MACD 柱狀圖低點卻比前次更高
→ 賣壓衰竭，底部反轉訊號（尤其配合 RSI 背離時威力倍增）

**空頭背離**：股價創新高，MACD 柱狀圖高點卻比前次更低
→ 買盤衰竭，頂部反轉訊號

### 短線操作應用
- MACD 不適合超短線（計算週期較長），最適合 1-4 週的中短線
- **最佳進場點**：股價在均線支撐 + MACD 柱狀圖由負轉正 + KD 黃金交叉三者共振

## 五、布林通道（Bollinger Bands）

### 組成說明
- **中軌**：20 日移動平均線（MA20）
- **上軌**：中軌 + 2 倍標準差（約 95% 的價格在此範圍）
- **下軌**：中軌 - 2 倍標準差
- **帶寬（Bandwidth）**：上下軌間距，反映波動率大小

### 核心應用

**布林收縮（Squeeze）**：帶寬縮小到歷史低位
- 代表市場即將出現大波動，方向未定
- 等待突破方向確認後跟進

**突破上軌**：
- **趨勢型行情**：突破上軌 + 量能放大 → 繼續持多，上軌為浮動阻力但非出場訊號
- **震盪型行情**：觸及上軌 → 短線過熱，可減倉

**突破下軌**：
- **趨勢型行情**：跌破下軌 + 量能放大 → 空頭延續，不能盲目抄底
- **震盪型行情**：觸及下軌 → 超跌，可短線布局反彈

**回歸中軌**：
- 股價長期偏離中軌（在上軌或下軌附近）→ 有回歸中軌的傾向
- 可用於均值回歸策略的出場點設定

### 帶寬判斷波動環境
- 帶寬窄（< 5%）→ 低波動，即將有方向選擇
- 帶寬寬（> 15%）→ 高波動，市場情緒激烈，風險較高

## 六、成交量分析

### 量價關係基本原則

| 量 | 價 | 意涵 |
|----|----|----|
| 放量 | 上漲 | 多方主導，健康上漲，趨勢可信 ✅ |
| 縮量 | 上漲 | 買盤不積極，上漲可信度低 ⚠️ |
| 放量 | 下跌 | 空方主導，趨勢性下跌，避免接刀 ❌ |
| 縮量 | 下跌 | 賣壓不大，下跌接近尾聲，可留意反彈 |
| 放量 | 橫盤 | 籌碼換手，方向即將選擇 |
| 地量 | 橫盤 | 市場極度冷清，往往是底部前兆 |

### 特殊量能訊號

**爆量長紅（巨量噴出）**：
- 短期強力做多訊號，但也可能是「噴出頂」
- 若爆量後次日縮量收紅 → 籌碼穩定，可持多
- 若爆量後次日縮量收黑 → 主力出貨，注意風險

**爆量長黑（巨量崩跌）**：
- 短期恐慌賣壓，若發生在支撐區 → 可能是洗盤底
- 若發生在高位 → 主力出貨，趨勢反轉訊號

**地量（成交量極度萎縮）**：
- 通常在長期下跌後出現
- 「地量見地價」→ 底部區域的重要訊號，可留意逆向做多機會

### 成交量 MA
- 成交量站上 20 日均量 → 放量，有效突破機率高
- 成交量跌破 20 日均量 → 縮量，趨勢動能減弱

## 七、K 線型態

### 反轉型態（Bottom Patterns）

#### 錘子線（Hammer）🔨
- **特徵**：下影線長（≥ 實體 2 倍），實體小，下影線在下方
- **出現位置**：下跌趨勢末端的支撐區
- **意涵**：盤中賣方把價格壓低，但收盤前多方強力反攻
- **操作**：次日開盤確認收紅後進場，停損設錘子線低點下方

#### 吞噬型態（Engulfing）
- **多頭吞噬**：大陽線完全包覆前一日陰線 → 強力反轉訊號
- **空頭吞噬**：大陰線完全包覆前一日陽線 → 強力頂部訊號
- 配合高成交量時可信度更高

#### 早晨之星（Morning Star）⭐
- **結構**：大陰線 + 小實體（星形）+ 大陽線，三日型態
- **出現位置**：下跌趨勢末端
- **意涵**：空方力道耗盡，多方接手
- **操作**：第三天確認後進場

#### 黃昏之星（Evening Star）
- **結構**：大陽線 + 小實體（星形）+ 大陰線
- **出現位置**：上漲趨勢末端
- **意涵**：多方力道耗盡，空方接手

### 持續型態（Continuation Patterns）

#### 十字線（Doji）
- **特徵**：開盤 = 收盤，形成十字形
- **意涵**：多空力量均衡，市場猶豫
- **解讀**：
  - 出現在上漲趨勢高位 → 警示訊號，可能反轉
  - 出現在下跌趨勢低位 → 反彈訊號
  - 出現在趨勢中段 → 可能是整理後繼續

#### 長上影線（Shooting Star / Hanging Man）
- **長上影線**：上影線長，代表高位遭到賣壓壓制
- 出現在高位 → 射擊之星，頂部警示
- 出現在低位 + 長下影線 → 錘子線，底部反彈訊號

### 型態的應用注意
- K 線型態需要**位置確認**：同樣的型態在支撐區有意義，在趨勢中段參考性較低
- 單一 K 線型態不如型態組合（3 根以上）可信度高
- 必須有量能配合：無量的 K 線型態訊號較弱

## 八、多指標共振操作框架

### 進場條件（做多）— 3 個以上共振才進場

| 條件 | 說明 |
|------|------|
| ✅ 位置 | 股價在重要支撐區（均線/前低/整數關卡）|
| ✅ 趨勢 | 均線多頭排列，或出現黃金交叉 |
| ✅ KD | 超賣區（<20）黃金交叉，或低位反彈 |
| ✅ RSI | 低位反彈，或多頭背離 |
| ✅ MACD | 柱狀圖由負轉正，或 MACD 黃金交叉 |
| ✅ 量能 | 放量上漲，或縮量整理後的放量突破 |
| ✅ K 線型態 | 出現底部反轉型態（錘子線/早晨之星/吞噬）|
| ✅ 籌碼面 | 法人買超（參考 tw-market.md 或 us-market.md）|

### 停損設定原則
- 短線（1-5 天）：停損設在進場 K 線低點下方 1-2%
- 中線（1-4 週）：停損設在最近重要低點或 20MA 下方
- 停損幅度不超過 5-8%（若支撐太遠，縮小部位或等更好的進場點）

### 出場原則
- **止盈出場**：達到目標價（前高/整數關卡/布林上軌）
- **訊號反轉出場**：出現空頭訊號（KD 死亡交叉/MACD 死叉/跌破均線）
- **時間出場**：持有超過預定天數但未達目標，評估是否繼續

## 九、技術分析的局限與注意事項

1. **技術指標是落後指標**：它們基於過去的價格，無法預測突發事件（財報、政策、黑天鵝）
2. **不同週期可能有矛盾**：日線看多，但週線可能在空頭趨勢，要確認週期一致
3. **量不大的個股技術面失真**：低流動性股票的技術型態參考性較低
4. **主力可以製造假訊號**：刻意做出技術型態誘多/誘空，所以要配合籌碼面驗證
5. **大環境優先**：市場整體空頭時，即使個股技術面好，也容易被拖累
